## 自建节点常用脚本
#### Debian/Ubuntu TCP拥塞控制技术 ——TCP-BBR 一键安装脚本
```
wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/bbr.sh && chmod +x bbr.sh && bash bbr.sh
```
#### V2ray一键安装脚本
```

```
